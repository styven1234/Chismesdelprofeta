# 🔥 Chismes del Profeta

**El mejor té del momento** - Un sitio web moderno de chismes, noticias escandalosas y trending topics.

## 🌟 Características

- 📱 Diseño responsive y moderno
- 🎯 Filtrado por categorías (Celebridades, Política, Deportes, Influencers)
- 🔥 Sección de Trending en tiempo real
- ❤️ Sistema de likes y reacciones
- 💬 Comentarios en cada publicación
- 📤 Opción de compartir
- 🎨 Interfaz atractiva con gradientes y animaciones

## 📂 Estructura del Proyecto

```
chismesdelprofeta/
├── index.html      # Página principal
├── styles.css      # Estilos y diseño
├── script.js       # Lógica interactiva
└── README.md       # Este archivo
```

## 🚀 Cómo usar

1. Abre `index.html` en tu navegador
2. Navega entre categorías
3. Interactúa con los posts: like, comentarios y compartir
4. Consulta el trending para ver los chismes más populares

## 🎨 Tecnologías

- **HTML5** - Estructura semántica
- **CSS3** - Estilos modernos con gradientes y animaciones
- **Vanilla JavaScript** - Interactividad sin dependencias

## 📝 Contenido

Actualmente incluye 6 chismes de ejemplo en diferentes categorías.
Puedes agregar más chismes modificando el array `chismes` en `script.js`.

## 🌐 Hosting

Este sitio está optimizado para desplegarse en **GitHub Pages**.

### Para activar GitHub Pages:

1. Ve a **Settings** → **Pages**
2. En "Source" selecciona **main branch**
3. ¡Listo! Tu sitio estará en: `https://styven1234.github.io/chismesdelprofeta`
